วิธีติดตั้งภาษา Python

ให้เขียนได้บนโปรแกรม Visual Studio Code

สำหรับโปรแกรมที่ต้องติดตั้ง มี 2 โปรแกรม คือ

1.Visual Studio Code
2.Python 3

วิธีการติดตั้ง Python 3 บน VS Code

1.ให้สร้างโฟลเดอร์สำหรับเก็บไฟล์ Code ของเราไว้สักที่ในคอมพิวเตอร์ เช่น สร้างโฟลเดอร์ที่ชื่อว่า py ไว้บน Desktop จากนั่นให้เปิดโปรแกรม VS Code ขึ้นมา แล้วให้เลือก Open Folder … “ที่เราสร้างไว้ คือ py”

วิธีการเปิดโฟลเดอร์เพื่อสร้างโปรเจค ให้เรากด Open Folder … หรือ ไปที่ File > Open Folder ตามรูปด้านล่าง

2.สร้างไฟล์ “ชื่ออะไรก็ได้หนึ่งไฟล์ ให้นามสกุล เป็น .py” จากนั่นเขียนคำสั่ง :

เมื่อเรา Save ไฟล์แล้ว จะมี หน้าต่างถามเราว่าต้องการติดตั้ง Extension for Python หรือไม่ ให้เราเลือก Install แต่ถ้าสำหรับคนที่ไม่มีหน้าต่างนี้ขึ้นให้ทำตามด้านล่าง

วิธีติดตั้ง Extensions สำหรับ Python
- ให้เราไปแถบด้านซายเลือก Extensions หรือกด Ctrl + Shift + X
- จากนั่นค้นหา Python และทำการติดตั้ง (เสร็จการติดตั้ง)
